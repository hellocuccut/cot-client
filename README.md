# Project Description
This project is a Minecraft client with various features to enhance gameplay.

## Features
- **Kill Aura**: Automatically attacks nearby enemies.
- **Flight**: Allows the player to fly freely in the game.
- **Speed**: Increases player movement speed.
- **Aimbot**: Automatically aims at enemies.
- **ESP**: Displays enemy locations through walls.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/hellocuccut/cot-client.git
   ```
2. Navigate into the project directory:
   ```bash
   cd cot-client
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage Guide
1. Start the client:
   ```bash
   npm start
   ```
2. Configure the desired features in the settings menu.
3. Enjoy the enhanced gameplay experience!
